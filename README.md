# УКАЗАНИЯ ПО РАБОТЕ С JETBOT_MIREA

## ИТАК, ВЫ РЕШИЛИ РАБОТАТЬ С JETBOT…

РТУ МИРЭА ИИИ КПУ, Грачев А.В., Зенина А.А.

---

Было написано некотрое колличество инструкций, для работы с jetbot_mirea, они реализуют следующие кейсы
1. Настройка ПИД-регуляторов на моторах.
2. Управление движением реального робота с клавиутуры.
3. Запус симуляции jetbot_mirea в Gazebo с эмуляцией датчиков.
4. Запуск картографировния и невигации на реальном роботе.
5. Запуск карттографирования и навигации для симуляции робота.

## Структура материала

Далее написан порядок действий для реализации кажого из кейсов.
Все материалы расположены в папке `docs/`.
Мы предполагаем, что вы развернули на компьютере, на котром работает или производите с которого подключение к роботу наш репозиторий по иструкции
**«0 Развёртывание репозитория с ПО для jetbot_mirea»** 

1. Настройка ПИД-регуляторов на моторах.
	1. «0 Загрузка прошивки на ESP32»;
	2. «1 Создание симлинка для ESP32»;
	3. «2 Настройка ПИД регуляторов моторов»;

2. Управление движением реального робота с клавиутуры.
	1. «0 Подключение к Jetson Nano»;
	2. **На роботе:** «1 Создание симлинка для ESP32»
	3. **На роботе:** «0 Развёртывание репозитория с ПО для jetbot_mirea»;
	4. «2 Запуск датчиков на роботе jetbot_mirea»;
	5. «3 Управление движением jetbot_mirea с помощью клавиатуры»;

3. Запус симуляции jetbot_mirea в Gazebo с эмуляцией датчиков.
	1. «1 Запуск симуляции в Gazebo»;

4. Запуск карттографирования и навигации по карте для симуляции робота.
	1. «3 Как запустить режим картографирования для симуляции»;

5. Запуск картографировния и невигации на реальном роботе.
	1. «3 Как запустить режим картографирования для реального робота»; *(находится в разработке)*
	2. «4 Как запустить режим автономной навигации по карте для реального робота»; *(находится в разработке)*

В папке `docs/` так же есть так же справочная инсформация. Это обширные документы, которые разъясняют работу робота, его конструкцию и дают подробности о написанном ПО для ROS2.

Всего 3 документа:

1.  **Аппаратные возможности** - описание всех использующихся модулей в jetbot_mirea, их характеристики, схема подключения и ссылки на дополнительные материалы. *(находится в разработке)*
2.  **Драйвера** - ПО, которое напрямую обращается в своей работе к аппаратной части робота, т.е. лидару, камере, ESP32, моторам. *(находится в разработке)*
3.  **Кроссплатформенное ПО** - ПО, которое не создавалось напрямую для Jetbot'a и его модулей, но настроено под jetbot_mirea. Там освещены такие темы как запуск стека навигации nav2. *(находится в разработке)*

## Сведения о программном обеспечении

ПО разрабатывалось для ROS2 Humble под Linux Ubuntu 22.04.
На jetbot_mirea (на вычислителе Jetson Nano) установлен ROS2 Galactic под Linux Ubuntu 20.04.

Вся написанная документация рассчитана под читателя, ознакомленного с фреймворком ROS2. Т.е. в повествовании опущено объяснение работы команд принадлежащих этому фреймворку.

Существует бесплатный курс на русском языке, в котором охватывается необходимый минимум материала для работы с ROS2. Ссылка на курс прилагается:
**ROS2 — Введение в робототехнику.**
