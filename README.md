## Введение

В этом документе представлены указания по использованию обширных материалов, разработанных для работы с jetbot_mirea.

## Возможности

Существующее программное обеспечение позволяет реализовать следующие кейсы:

1.  Настройка ПИД-регуляторов моторов по скорости с помощью графического интерфейса.
2.  Одновременный корректный запуск датчиков с получением таких топиков как /scan, /odom. (запуск лидара и получение информации о скоростях и одометрии с ESP32, т. е. с энкодеров моторов).
3.  Запуск симуляции jetbot_mirea в Gazebo с эмуляцией работы датчиков (лидар, обе камеры, IMU, одометрия, GPS) и визуализацией в rviz.
4.  Запуск стека nav2: картографирования, локализации и навигации для реального робота и для его симуляции.

## Структура материала

Все материалы расположены в папке `docs/` . Их можно разделить на 2 группы:

-   Подробное описание jetbot_mirea, его аппаратного и программного обеспечения.
-   Короткие инструкции по установке и запуску программного обеспечения.

Подробное описание jetbot_mirea состоит из 3-ёх частей:

1.  **Аппаратные возможности** - описание всех использующихся модулей в jetbot_mirea, их характеристики, схема подключения и ссылки на дополнительные материалы. *(находится в разработке)*
2.  **Драйвера** - ПО, которое напрямую обращается в своей работе к аппаратной части робота, т.е. лидару, камере, ESP32, моторам. *(находится в разработке)*
3.  **Кроссплатформенное ПО** - ПО, которое не создавалось напрямую для Jetbot'a и его модулей, но настроено под jetbot_mirea. Там освещены такие темы как запуск стека навигации nav2. *(находится в разработке)*

Короткие инструкции разделены по уровню вложенности.
Если имя файла начинается с «0\_», то это инструкция, которую можно выполнить, не ссылаясь на другие инструкции. Если имя файла начинается с «1\_», то это инструкция, которую можно выполнить, ссылаясь только на инструкцию с префиксом «0\_» и так далее.

## Сведения о программном обеспечении

ПО разрабатывалось для ROS2 Humble под Linux Ubuntu 22.04.
На jetbot_mirea (на вычислителе Jetson Nano) установлен ROS2 Galactic под Linux Ubuntu 20.04.

Вся написанная документация рассчитана под читателя, ознакомленного с фреймворком ROS2. Т.е. в повествовании опущено объяснение работы команд принадлежащих этому фреймворку.

Существует бесплатный курс на русском языке, в котором охватывается необходимый минимум материала для работы с ROS2. Ссылка на курс прилагается:
**ROS2 — Введение в робототехнику.**

## Как использовать «Короткие инструкции»

Находите инструкцию с нужным названием и пошагово выполняете её.

На данный момент доступны следующие короткие инструкции:

-   «0 Загрузка прошивки на ESP32»;
-   «0 Подключение к Jetson Nano»;
-   «0 Развёртывание репозитория с ПО для jetbot_mirea»;
-   «1 Запуск симуляции в Gazebo»;
-   «1 Создание симлинка для ESP32»;
-   «2 Запуск датчиков на роботе jetbot_mirea»;
-   «2 Настройка ПИД регуляторов моторов»;
-   «3 Как запустить режим картографирования для реального робота»; *(находится в разработке)*
-   «3 Как запустить режим картографирования для симуляции»;
-   «3 Управление движением jetbot_mirea с помощью клавиатуры»;
-   «4 Как запустить режим автономной навигации по карте для реального робота»; *(находится в разработке)*
-   «4 Как запустить режим автономной навигации по карте для симуляции».
