# ESP32 Serial Bridge GUI

Программа с графическим интерфейсом для управления роботом на базе ESP32 через последовательный порт. Предоставляет интуитивный интерфейс для отправки команд управления, настройки PID-регуляторов и визуализации данных в реальном времени.

## Основные возможности

### 🎮 Управление движением
- **Velocity Control**: Управление линейной и угловой скоростью
- **Wheel Control**: Прямое управление скоростями левого и правого колес
- **Экстренная остановка**: Красная кнопка STOP для немедленной остановки робота

### ⚙️ Настройка PID-регуляторов
- Настройка коэффициентов PID для левого и правого двигателей
- Просмотр текущих установленных значений PID
- Отправка и установка параметров регуляторов

### 📊 Визуализация данных
- **В реальном времени**: Графики обновляются с интервалом 300мс
- **Мульти-вкладки**: Отдельные графики для каждой категории данных
- **Полный обзор**: Вкладка со всеми графиками одновременно

### Отслеживаемые параметры
- Линейная и угловая скорости платформы
- Скорости левого и правого колес

## Установка и запуск
1. Установите все python-зависимости:
```bash
sudo apt install python3-pip python3-tk
pip install -r pip_requirements.txt
```
2. В терминале выполните:
```bash 
python3 fast_gui.py
```

### Настройка порта
По умолчанию используется `/dev/esp32`. вы можете изменить это. Для изменения порта отредактируйте переменную `serial_port` в функции `main()` скрипта `determine_id.bash`.

## Интерфейс
- **Красная кнопка STOP** - экстренная остановка
  
### Вкладка "Velocity Control"
- Поля ввода: linear, angular
- Кнопка "Send Velocity"

- Графики: v_linear_x(t), v_angular_z(t)

### Вкладка "Wheel Control" 
- Поля ввода: left, right
- Кнопка "Send Wheel"
- Графики: v_left(t), v_right(t)

### Вкладка "PID Tuning"
- Настройки коэффициентов: Kp_L, Ki_L, Kd_L, Kp_R, Ki_R, Kd_R
- Кнопки: "Send PID", "Get PID"
- Отображение текущих значений
- Графики скоростей колес

### Вкладка "All Plots"
- 4 графика на одном экране:
  - v_linear_x(t), v_angular_z(t) 
  - v_left(t), v_right(t)

## Форматы сообщений

### Отправляемые команды
```
$1;linear;angular;#     - Управление скоростью
$2;left;right;#         - Управление колесами  
$3;Kp_L;Ki_L;Kd_L;Kp_R;Ki_R;Kd_R;#  - Настройка PID
$4;Kp_L;Ki_L;Kd_L;Kp_R;Ki_R;Kd_R;#  - Настройка PID с записью в постоянную память
$5;#                    - Запрос текущих PID
```

### Принимаемые данные
```
$1;x;y;theta;v_linear_x;v_angular_z;v_left;v_right;# - Позиция, скорости
$2;Kp_L;Ki_L;Kd_L;Kp_R;Ki_R;Kd_R;#  - Текущие PID
```
