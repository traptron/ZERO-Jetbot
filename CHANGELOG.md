## [1.0.0] - 2026-03-04

### Добавлено

#### Новый пакет: `teleop_jetbot`

**Описание:** Пакет для упрощенного телеуправления роботом JetBot с клавиатуры через локальную сеть.

**Компоненты:**
- `teleop_keyboard.py` - узел для управления с клавиатуры (WASD, QE, X)
- `teleop_pc.launch.py` - launch файл для ПК
- `teleop_robot.launch.py` - launch файл для робота
- Полная документация в `teleop_jetbot/README.md`

**Возможности:**
- Управление роботом с клавиатуры компьютера
- Работа через ROS2 топики `/cmd_vel` по локальной сети
- Автоматическая конвертация в протокол ESP32 через существующий `serial_bridge_package`
- Совместимость: ROS2 Humble (ПК) ↔ ROS2 Galactic (робот)
- Плавное управление скоростью (инкрементное изменение)
- Защита от ошибок при запуске без интерактивного терминала
- Автоматическая остановка робота при завершении программы

**Технические детали:**
- Максимальная линейная скорость: 0.5 м/с
- Максимальная угловая скорость: 1.5 рад/с
- Частота публикации команд: ~10 Гц
- Интеграция с аппаратной защитой ESP32 (таймаут 5 сек)
